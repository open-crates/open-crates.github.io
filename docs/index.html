Docs

Starter documentation for Open Crates. We’ll expand this as the first beta ships.

Getting started

  1. Install Open Crates (download link will be added).
  2. Open the app and choose a music service to connect.
  3. Authorize access in your browser (OAuth).
  4. Select a playlist and export a Serato crate.

Tip: Keep your Serato library backed up before importing new crates.

Connecting services

  • Spotify: OAuth login (PKCE). No client secret is stored in the app.
  • YouTube: OAuth login. Exports will map playlist entries to searchable track metadata where possible.
  • Apple Music: May require a short-lived developer token issued by a minimal backend service.

Export behavior

  • Exports create a Serato crate file for the selected playlist.
  • Track matching depends on available metadata from each platform.
  • If a track can’t be matched, it will be listed in the export report.

Troubleshooting

  • Login loop: try disconnecting and reconnecting the service.
  • Missing tracks: some platforms don’t expose full metadata; try searching by title/artist.
  • Serato can’t see crate: confirm the crate was exported into the correct Serato library folder.

If you hit a bug, open an issue on GitHub with your OS + a short description.
